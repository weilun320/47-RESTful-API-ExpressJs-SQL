Welcome to the API!

How to use this API:

  • GET /posts : Get all posts. Retrieve all posts.
  • GET /posts/:id : Get a post by ID. Replace ":id" with the ID of the post you want to retrieve.
  • GET /posts/author/:authorName : Get all posts by author. Replace ":authorName" with the author of the posts you want to retrieve.
  • GET /posts/dates/:startDate/:endDate : Get all posts in a certain date range. Replace ":startDate" with the starting date and ":endDate" with the ending date of the posts you want to retrieve.
  • POST /posts : Create a new post. The body should be a JSON object with "title", "content", and "author" keys.
  • PUT /posts/:id : Update an existing post. Replace ":id" with the ID of the post you want to update. The body should be a JSON object with "title", "content", and "author" keys you want to update.
  • DELETE /posts/:id : Delete a post. Replace ":id" with the ID of the post you want to delete.
  • DELETE /posts/author/:authorName : Delete all posts by author. Replace ":authorName" with the author of the posts you want to delete.
